# VI比較資料

対象資料:
- 案1『森の木漏れ日』
- 案2『おひるねの夢』
- 案3『しっぽの足跡』

## 1. 全体サマリー

3案はいずれも「やさしさ」や「安心感」をベースにしつつ、表現の方向性が大きく異なります。

- 案1『森の木漏れ日』は、自然素材・余白・手触り感を軸にした、最も王道で安定感のある案です。
- 案2『おひるねの夢』は、休息・浮遊感・境界の曖昧さを軸にした、最も幻想的で感情訴求の強い案です。
- 案3『しっぽの足跡』は、不揃いさ・遊び心・手作業感を軸にした、最も個性的で記憶に残りやすい案です。

## 2. 比較表

| 比較項目 | 案1『森の木漏れ日』 | 案2『おひるねの夢』 | 案3『しっぽの足跡』 |
|---|---|---|---|
| コンセプトの核 | 素朴な手触りと呼吸する余白 | 輪郭のやさしい夢心地のグラデーション | 整えすぎないリズムのある不揃い |
| ブランド印象 | あたたかい、自然体、上品、安心 | やわらかい、幻想的、静か、包容的 | 親しみやすい、自由、チャーミング、活発 |
| 感情価値 | 緊張をゆるめる、ほっとする | 深く休める、包まれる、現実から離れる | 力を抜ける、愛着が湧く、笑顔になる |
| デザインの特徴 | 余白を広く取り、自然なゆらぎを活かす | 境界をぼかし、にじみや浮遊感を強調 | ズレや不揃いを意図的に活かす |
| 色の方向性 | ブラウン、ベージュ、グリーン | ラベンダー、白、コーラル | テラコッタ、ミルクティー、黒 |
| 書体の印象 | 丸みと情緒のある、読みやすい構成 | 細く軽く、空気感のある構成 | 手書き感やにじみを活かした構成 |
| 写真表現 | 逆光、暖色、静かな余白のある構図 | ハイキー、甘いピント、白昼夢の質感 | スナップ的で生活感のある切り取り |
| イラスト表現 | 鉛筆線や塗りムラのある手描き | 水彩のにじみ、境界のない表現 | 切り絵、版画、ラフで愛嬌のある表現 |
| Webとの相性 | 落ち着いたブランドサイトと好相性 | 世界観重視の演出型サイトと好相性 | 小さな驚きや遊びのあるサイトと好相性 |
| 印刷物との相性 | 再生紙やクラフト紙など素材提案が強い | トレペや重なり表現で独自性を出しやすい | 冊子やタグなど手作業感の演出に強い |
| 強み | 安定感、共感の得やすさ、展開のしやすさ | 世界観の独自性、癒やし訴求、没入感 | 記憶定着、親近感、ブランドのキャラクター化 |
| 懸念点 | やや無難に見える可能性 | 好みが分かれやすく、実装難度がやや高い | 統制を誤ると雑多に見える可能性 |

## 3. それぞれの向いている判断軸

### 案1『森の木漏れ日』が向いている場合

- 初見でも安心感が伝わるブランドにしたい
- ナチュラルさや誠実さを重視したい
- Web、紙、パッケージへ横展開しやすい案を優先したい
- 長く使える普遍性を重視したい

### 案2『おひるねの夢』が向いている場合

- 「休息」や「癒やし」を最も強く打ち出したい
- 他案よりも情緒的、感覚的な世界観を重視したい
- ビジュアル演出で没入感を作りたい
- やさしさの中に少し非日常感を持たせたい

### 案3『しっぽの足跡』が向いている場合

- ブランドに遊び心や人格を持たせたい
- 動物らしさ、手作業感、親しみやすさを強く出したい
- 多少ラフでも記憶に残る個性を優先したい
- SNSや店頭での印象的な見え方を重視したい

## 4. ひとことで言うなら

- 案1: 「いちばん安心して選びやすい、自然体の王道案」
- 案2: 「いちばん深く休めそうな、夢見心地の世界観案」
- 案3: 「いちばん愛着が湧きやすい、遊び心のある個性派案」

## 5. 比較検討時のおすすめ観点

選定時は、単に見た目の好みだけでなく、次の観点で判断すると整理しやすいです。

- ブランドとして最も伝えたい価値は何か
- 初見の人に「安心」「癒やし」「親しみ」のどれを最優先で感じてほしいか
- Webや印刷、パッケージで継続運用したときに最も再現しやすいのはどれか
- 将来的にブランドの個性をどこまで強く打ち出したいか

## 6. 参考としての一次推奨

もし「まず軸となる1案を選び、その後に磨き込みたい」という進め方であれば、バランス面では案1『森の木漏れ日』が最も起点にしやすいです。

理由:
- 自然、ぬくもり、余白という価値が直感的に伝わりやすい
- Web、紙、パッケージの各媒体へ素直に展開しやすい
- 過度に尖りすぎず、関係者間で合意形成しやすい

一方で、ブランドの差別化や記憶定着を最優先するなら案3、休息感という情緒価値を最優先するなら案2が強い候補になります。
