# VIマニュアル案 3：『しっぽの足跡』案

## 1. デザインコンセプトの言語化
**「整えすぎない、リズムのある不揃い」**
少しの失敗やズレを「チャームポイント」として肯定する、遊び心のあるトーン＆マナーです。あえてグリッドから少しずらしたり、切り貼りのようなコラージュ感を取り入れたりすることで、「愛される不完全さ」を視覚化します。

## 2. ロゴの使用ルール
- **アイソレーション（保護空間）：** 文字の「ハネ」や「はみ出し」を許容する、緩やかな保護空間を設定します。
- **最小使用サイズ：** 10mm（シンボルマークのみの場合はさらに縮小可）。
- **禁止事項：** 水平垂直にこだわりすぎる配置。5度程度の傾きは「味」としてむしろ推奨されます。

## 3. カラーパレットの定義
- **メインカラー：** 泥んこテラコッタ（#A0522D） - 親しみやすさと土の匂い。
- **サブカラー：** ミルクティー色（#E6D5C3） - 全体を馴染ませる調和の色。
- **アクセントカラー：** しっぽの黒（#2C2C2C） - デザインを適度に引き締める「点」の色。

## 4. タイポグラフィ（書体）
- **見出し用：** 自作風の手書きフォント、または凸版文久見出し明朝（味のある墨だまり）。
- **本文用：** 秀英にじみ丸ゴシック - インクの滲みを再現した、体温の伝わる書体。

## 5. イメージ・イラスト・写真のトーン
- **写真：** 動物のアップや、食べかけのクッキーなど「生活の断片」を切り取った、着飾らないスナップ写真。
- **イラスト：** 切り絵や版画のような、力強くも愛嬌のある不揃いなタッチ。

## 6. 媒体別の展開イメージ
- **Webサイト：** ボタンの形が一つひとつ微妙に異なる、ハンドメイド感のあるボタンやバナー。
- **パンフレット・資料：** あえて端を裁断せず、不揃いな束感（アンカット）を出したワイルドな冊子。
- **製品・パッケージ：** 麻紐や手書きのタグ、端切れを使った、贈る人の体温が伝わる手作業感のある梱包。
