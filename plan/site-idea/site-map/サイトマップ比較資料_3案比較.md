# サイトマップ比較資料

対象資料:
- 案1【世界観没入型】物語を読み進める体験プラン
- 案2【コミュニティ重視型】常連の輪に加わる安心プラン
- 案3【実用・休息特化型】効率社会からの避難所プラン

## 1. 全体サマリー

3案はいずれも「ひるね日和」の魅力を伝える構成ですが、何を最優先で伝えるかが異なります。

- 案1は、世界観や物語性を通じてブランドに没入してもらう構成です。
- 案2は、動物スタッフや日常の出来事を通じて親近感とファン化を促す構成です。
- 案3は、初来店前の不安を減らし、必要な情報へ迷わずたどり着ける構成です。

## 2. 比較表

| 比較項目 | 案1【世界観没入型】 | 案2【コミュニティ重視型】 | 案3【実用・休息特化型】 |
|---|---|---|---|
| 設計思想 | 物語を読み進める体験 | 常連の輪に加わる安心感 | 迷わず使える休息導線 |
| 最初に伝わる価値 | 世界観、空気感、ブランドの詩情 | 親しみ、スタッフの個性、場の温度 | 安心、分かりやすさ、休みやすさ |
| 主役になるコンテンツ | コンセプト、過ごし方、きまぐれメニュー | 店員紹介、日和日記、SNS連動 | はじめての方へ、設備案内、最新状況 |
| 情報構造の特徴 | 情緒的なネーミングで没入感を強化 | 人や出来事を起点に回遊を促進 | 用件別に整理され、目的到達が速い |
| UXの指針 | 誘う（Invite） | 案内する（Accompany） | 迷わせない、急かさない |
| ナビゲーションの印象 | 森の奥へ進むような探索型 | スタッフに導かれる伴走型 | 必要情報へ素直に届く整理型 |
| 来店動機の作り方 | 今の営業状況と雰囲気で衝動来店を促す | 日常接触を増やしファン化から来店へつなぐ | 混雑・営業・アクセス情報で来店判断を支える |
| リピート施策 | 日替わりメニューによる再訪動機 | ブログやSNSによる継続接触 | 来店を習慣化するカレンダー導線 |
| 初見ユーザーとの相性 | 高いが、世界観理解に少し時間がかかる | 高いが、キャラクター訴求への好みが分かれる | 最も高い。迷いや不安を解消しやすい |
| ブランド表現の強さ | 強い | 強い | 中〜強 |
| 運用負荷 | 中 | 高め | 中 |
| 強み | 没入感、独自性、ブランド体験の濃さ | 親近感、ファン化、SNSとの親和性 | 分かりやすさ、実用性、コンバージョンの安定感 |
| 懸念点 | 情報の即時性より雰囲気が先行しやすい | 継続更新が前提で、運用体制が必要 | 世界観の意外性や記憶性は相対的に控えめ |

## 3. 各案の向いているケース

### 案1【世界観没入型】が向いている場合

- ブランドの空気感や物語を最優先で伝えたい
- ただの店舗情報ではなく、訪問前から体験を始めたい
- 写真、動画、コピーで感情を動かすサイトにしたい
- 世界観の強いカフェとして印象を残したい

### 案2【コミュニティ重視型】が向いている場合

- 動物スタッフや日常の出来事をブランド資産にしたい
- リピーターやファンコミュニティを育てたい
- SNSと連動して接触頻度を高めたい
- 「誰に会いに行く場所か」を明確にしたい

### 案3【実用・休息特化型】が向いている場合

- はじめて来る人の不安を最小化したい
- 営業状況、席、ルール、アクセスを分かりやすく届けたい
- 来店判断をスムーズにしたい
- 実用性とブランドらしさのバランスを重視したい

## 4. ひとことで言うなら

- 案1: 「ブランドの世界観に浸ってから来店したくなる案」
- 案2: 「お店やスタッフを好きになって通いたくなる案」
- 案3: 「安心して迷わず来店できる、実用性の高い案」

## 5. 比較検討時のおすすめ観点

選定時は、見た目の好みだけでなく、次の観点で比較すると判断しやすいです。

- 新規来店者に最初に何を感じてほしいか
- サイトの役割を「世界観体験」「ファン化」「来店支援」のどこに置くか
- 更新頻度や運用体制にどこまで対応できるか
- SNS、ブログ、最新情報など継続発信をどこまで重視するか
- ブランドの個性と、情報の分かりやすさのどちらを優先するか

## 6. 参考としての一次推奨

もし「まず実店舗への導線として機能しつつ、ブランドの魅力もきちんと伝えたい」という進め方であれば、全体バランスでは案3【実用・休息特化型】が最も起点にしやすいです。

理由:
- 初見ユーザーが必要とする情報に最短で到達しやすい
- 「はじめての方へ」と「設備案内」が来店前の不安を解消しやすい
- 休息を提供する場所としての価値が、機能面と情緒面の両方で整理されている

一方で、ブランドの物語性を最優先するなら案1、ファン化やコミュニティ形成を最優先するなら案2が強い候補になります。
