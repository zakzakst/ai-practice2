# サイトマップ案1：【世界観没入型】物語を読み進める体験プラン

## 1. サイトの全体構造
- **HOME：ひるね日和の朝（メインビジュアル）**
    - 店内の空気感を伝える動画・画像
    - 現在の営業状況（リアルタイム表示）
- **コンセプト：森の隣人たちからの手紙**
    - **私たちの想い**：完璧を求めない不完全さへの愛
    - **店長からのごあいさつ**：動物店長によるゆるいメッセージ
- **おしながき：心とお腹を満たすもの**
    - **飲みもの**：木の実ブレンド、夜ふかしブレンド等
    - **食べもの**：はちみつ森のパンケーキ、どんぐり風クッキー等
    - **本日のきまぐれ**：日替わりの一品（SNS連動更新）
- **過ごし方：ひるねの作法（ルール）**
    - **お席の案内**：止まり木、丸太、ソファ席の紹介
    - **ひるね日和の約束**：しっぽ注意、おひるね中は静かに等のルール
- **アクセス：三日月坂のふもとへ**
    - 地図・交通案内・営業時間
    - 臨時サマー営業（店長が暑い日）の履歴と基準

## 2. 各ページの役割とコンセプトの紐付け
- **過ごし方ページ**：「完璧を求めない、ありのままの休息を保証します」というブランドの約束を伝える最重要ページ。客が店側に合わせることで生まれる独特の連帯感を醸成します。
- **コンセプトページ**：「ポンコツで愛らしい運営スタイル」を肯定的に伝え、デジタル・効率社会で疲れたユーザーの緊張を解く役割を担います。

## 3. ユーザー体験（UX）とナビゲーションの指針
- **指針**：「指示するのではなく、誘う（Invite）」。
- **設計**：カチッとしたボタンではなく、手書き風のアイコンや、スクロールに合わせて動物たちが歩き回るようなアニメーションを配置。次のページへ「進む」のではなく「森の奥へ歩く」ような感覚を提供します。

## 4. コンバージョン（成果）への導線設計
- **導線**：全ページのフッターに「今のひるね状況（営業中／おやすみ中）」を表示。
- **行動**：「今、空いているなら行ってみよう」という突発的な来店動機を促します。

## 5. コンテンツ制作の優先順位（逆ピラミッド型）
1. **ひるねの作法（ルール）**：最もユニークなUSPであり、心理的ハードルを下げるために最優先で伝えます。
2. **本日のきまぐれメニュー**：訪れるたびに変化があることを示し、リピート意欲を高めます。
3. **アクセス情報**：実店舗への誘導として不可欠な情報を最後に配置します。
