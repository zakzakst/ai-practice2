# キャラ情報

## 1. 店長・シロクマ

- 名前：白川 しろまる
- ニックネーム：しろまる店長
- 動物の学名：*Ursus maritimus*
- 動物の分類：哺乳綱 食肉目 クマ科
- 好きな食べ物：はちみつパンケーキ、冷たいミルク、鮭サンド
- 苦手な食べ物：激辛料理、熱すぎるスープ
- 好きなこと：店の理念を語ること、涼しい場所で休むこと、看板メニューを考えること
- 性格：理想家で面倒見がよく、頼りになりそうで少し抜けている
- 口ぐせ：「うちは“癒やし”が売りだからね」
- 接客スタイル：最初は頼もしいが、暑い日は急に戦力外になる
- ギャグの軸：立派なことを言った直後にへばる

## 2. ホール係・ネコ

- 名前：三毛野 こまり
- ニックネーム：こまちゃん
- 動物の学名：*Felis catus*
- 動物の分類：哺乳綱 食肉目 ネコ科
- 好きな食べ物：クリーム系スイーツ、白身魚、ミルクプリン
- 苦手な食べ物：柑橘系、刺激の強い香辛料
- 好きなこと：日向ぼっこ、気に入ったお客さんへの特別サービス、かわいい小物集め
- 性格：気まぐれで要領がよく、褒められると機嫌がよくなる
- 口ぐせ：「今日はそういう気分じゃないの」
- 接客スタイル：好きな客には完璧、そうでないと省エネ
- ギャグの軸：プロ級の接客力が機嫌で大きく変わる

## 3. バリスタ・フクロウ

- 名前：月見里 ふくろう
- ニックネーム：フク
- 動物の学名：*Strix uralensis*
- 動物の分類：鳥綱 フクロウ目 フクロウ科
- 好きな食べ物：深煎りコーヒー、ナッツ、ビターチョコ
- 苦手な食べ物：朝食向けの軽すぎるメニュー、甘すぎる飲み物
- 好きなこと：豆の研究、静かな夜の仕込み、うんちくを語ること
- 性格：知的でマイペース、夜になると急に元気
- 口ぐせ：「その抽出には理論があります」
- 接客スタイル：説明は丁寧だが長い
- ギャグの軸：昼は半分寝ているのに、コーヒーの話になると止まらない

## 4. パティシエ・リス

- 名前：木ノ実 りん
- ニックネーム：りんりん
- 動物の学名：*Sciurus lis*
- 動物の分類：哺乳綱 げっ歯目 リス科
- 好きな食べ物：ナッツ、クッキー、ベリータルト
- 苦手な食べ物：苦すぎるコーヒー、汁気の多い料理
- 好きなこと：お菓子作り、材料集め、かわいい包装を考えること
- 性格：働き者でせっかち、少しそそっかしい
- 口ぐせ：「あれ？ ここに置いたのに！」
- 接客スタイル：厨房特化型で、表に出ると少し早口になる
- ギャグの軸：隠した食材や道具を自分で見失う

## 5. 新人・イヌ

- 名前：犬飼 たろ
- ニックネーム：たろちゃん
- 動物の学名：*Canis lupus familiaris*
- 動物の分類：哺乳綱 食肉目 イヌ科
- 好きな食べ物：ハンバーグ、オムライス、ソーセージパン
- 苦手な食べ物：苦い野菜、香りの強すぎる発酵食品
- 好きなこと：誰かの手伝い、イベント企画、散歩と配達
- 性格：素直で元気、やる気はあるが空回りしがち
- 口ぐせ：「ぼく、やります！」
- 接客スタイル：笑顔は満点だが、段取りはまだ甘い
- ギャグの軸：善意で頑張るほど仕事が増える

## 6. 常連客・カピバラ

- 名前：湯川 かぴ夫
- ニックネーム：かぴさん
- 動物の学名：*Hydrochoerus hydrochaeris*
- 動物の分類：哺乳綱 げっ歯目 テンジクネズミ科
- 好きな食べ物：野菜たっぷりサンド、プリン、ぬるめの飲み物
- 苦手な食べ物：熱すぎる料理、せかされながら食べる食事
- 好きなこと：ぼんやり過ごすこと、店で本を読むこと、みんなの話を聞くこと
- 性格：穏やかで動じず、聞き上手
- 口ぐせ：「まあ、なんとかなるんじゃないですか」
- 立ち位置：客なのに、その場を落ち着かせるまとめ役になることが多い
- ギャグの軸：一番店員っぽく落ち着いているのにただの客
