# サイトマップ案3：【実用・休息特化型】効率社会からの避難所プラン

## 1. サイトの全体構造
- **HOME：深呼吸、ひとつ。**
    - 静謐な店内のビジュアル
    - 「おひるね推奨」のタグライン
- **はじめての方へ：ここは「正しくなくていい」場所です**
    - **ブランドコンセプト**：役割を脱ぎ捨てて素に戻るための場所
    - **独自のルール**：大きな声を出さない、しっぽに注意等のマナー
- **メニューと設備：休息の道具たち**
    - **お飲みもの・お食事**：木の実ブレンド、のんびりプリン等
    - **お席のタイプ**：体格や気分に合わせた選択（止まり木、ソファ等）
- **最新状況：今のひるね日和**
    - 現在の混雑状況、臨時営業のお知らせ
- **アクセス**
    - 三日月坂のふもとへの行き方（分かりやすさ重視）

## 2. 各ページの役割とコンセプトの紐付け
- **メニューと設備ページ**：「自分の体格や気分に合ったお気に入りの居場所を見つける」という機能的ベネフィットをダイレクトに提示します。
- **はじめての方へページ**：ビジネスパーソンや主婦など、日常で「しっかり者」を演じているターゲットに対し、ここが「解放区」であることを宣言します。

## 3. ユーザー体験（UX）とナビゲーションの指針
- **指針**：「迷わせない、急かさない（Efficiency with Calm）」。
- **設計**：目的のページへ最短で辿り着けるよう情報を整理しつつも、背景には落ち着いた「くすんだグリーン」や「木のブラウン」を配し、視覚的なノイズを徹底的に排除します。

## 4. コンバージョン（成果）への導線設計
- **導線**：アクセス情報の近くに「カレンダーに追加（定休日チェック）」ボタンを配置。
- **行動**：ユーザーが自分のスケジュールに「休息時間（ひるね日和への来店）」を予約する習慣を作ります。

## 5. コンテンツ制作の優先順位（逆ピラミッド型）
1. **お席のガイド（設備紹介）**：どのような空間で休めるのかという物理的な安心感を最優先。
2. **コンセプト（はじめての方へ）**：なぜこの店が必要なのかという情緒的価値の提示。
3. **メニュー・料金**：実務的な情報を最後にまとめます。
