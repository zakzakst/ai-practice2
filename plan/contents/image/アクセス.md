# 🧩 補足（実装メモ）

- 合計：11枚（写真系9枚 + イラスト2枚）
- ヒーロー画像 → 横長（16:9〜21:9）
- カード画像 → 1:1推奨
- 地図 → SVGまたは軽量画像推奨
- 夜景画像 → 少し彩度を落とすと全体トーンと統一しやすい
