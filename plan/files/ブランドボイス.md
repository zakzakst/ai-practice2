# ブランドボイス・ガイドライン：『ひるね日和（仮）』

本ガイドラインは、ブランドコンセプトである**「完璧を求められる日常からの解放区」**を言葉で体現するための指針です。動物たちの「愛される不完全さ」を核とし、読み手が「自分も不完全でいいんだ」と安心できるコミュニケーションを目指します。

---

## 1. ブランドボイスの定義（人格）
### **「一生懸命だけど、どこか抜けている森の隣人」**
ブランドを一人の人格に例えるなら、**「エプロンが少し傾いていることに気づかず、満面の笑みでお茶を淹れてくれる、おせっかいで優しい森の住人」**です。

* **誠実で一生懸命：** 適当に手を抜くのではなく、全力でおもてなしをしようとした結果の「ポンコツさ」が、相手の緊張を解く鍵となります。
* **フラットな関係性：** 上下関係や過剰な敬語で壁を作らず、同じ目線で「まあ、いいじゃない」と寄り添う包容力を持っています。
* **独自のペース：** 効率や正解を求める現代社会のペースではなく、季節の移ろいや動物の呼吸に合わせた「ゆっくりとした時間軸」を大切にします。

## 2. ボイスチャート（Do/Don'tリスト）

| 特性 | 意味 | 推奨する表現（Do） | 避けるべき表現（Don't） |
| :--- | :--- | :--- | :--- |
| **愛らしい不完全さ** | 完璧でないことを隠さず、むしろ親しみとして提示する | 「少し書き直してしまいましたが…」「今日のメニューは風の向くままに」 | 「寸分の狂いもない」「最高品質を保証」「完璧なオペレーション」 |
| **受容と肯定** | 相手の失敗や休息を丸ごと受け入れる | 「そのままのあなたで」「どうぞ、ゆっくりおひるねを」「失敗もスパイスです」 | 「〜すべきです」「正しく」「効率的に」「遅刻厳禁」 |
| **温度感のある言葉** | 記号的・機械的な言葉を避け、手触り感のある表現を使う | 「ぬくもり」「しっぽが紡ぐ」「ひたむき」「ふかふか」 | 「ソリューション」「最適化」「シナジー」「ユーザーベネフィット」 |
| **動物的な視点** | 人間中心のロジックではなく、生命のゆらぎを感じさせる | 「店長の機嫌が良いので」「日向ぼっこが似合う日」「丸太の感触」 | 「KPI」「戦略的」「市場シェア」「ロジカル」 |

## 3. 場面別のトーン調整ルール

### **■ 営業用プレゼン資料（信頼感・専門性）**
* **トーン：** 誠実・情熱的
* **調整：** 「なぜ不完全さが現代に必要か」を、独自の哲学として堂々と伝えます。
* **例：** 「私たちは、現代社会が失った『隙（スキ）』をデザインすることで、真のメンタルリカバリーを提供します。」

### **■ SNS投稿（親しみやすさ・会話体）**
* **トーン：** チャーミング・日常的
* **調整：** 動物たちの「今日の様子」を隣人が報告するように、柔らかい語尾で親密な距離感を演出します。
* **例：** 「今日の店長、お昼寝に夢中でメニューを出し忘れちゃいました。そんな日もありますよね。のんびりお待ちください。」

### **■ プレスリリース（誠実・客観的）**
* **トーン：** 丁寧・凛とした
* **調整：** 「解放区」という言葉を使い、既存のカフェビジネスとは一線を画す姿勢を客観的に示します。
* **例：** 「不揃いな日常を愛おしく包む場所として、動物たちが主導する新しい休息の形を提案します。」

### **■ 自社ブログ（共感・丁寧な解説）**
* **トーン：** 穏やか・内省的
* **調整：** 読者の「疲れ」を労い、物語を読み聞かせるような文体で、コンセプトの背景を深く掘り下げます。
* **例：** 「背負っていた荷物を、ここでは少しだけ下ろしてみませんか。しっぽの動きを眺めているうちに、不思議と呼吸が深くなるはずです。」

## 4. AI執筆用のライティング・ルール
AIに文章を書かせる際は、以下の指示をプロンプトに含めてください。

1. **「ひらがな」を意識的に混ぜる：** 硬さを取るため、平易な言葉を選んでください（例：休息→ひとやすみ、完璧→かんぺき）。
2. **五感を刺激する形容詞：** 「ぬくぬくした」「心地よい重み」など、手触り感のある言葉を使ってください。
3. **問いかけと余白：** 断定を避け、「〜ですね」「〜してみませんか」といった、相手の心にスペースを作る文末にしてください。
4. **主観的な視点：** 書き手の顔が見えるよう、「〜だと思っています」「〜を見守っています」という表現を使ってください。
