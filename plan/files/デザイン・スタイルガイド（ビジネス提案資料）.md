# デザイン・スタイルガイド（ビジネス提案資料）

## スライドデザインの基本方針
- トーン：プロフェッショナル・信頼感・モダン
- 余白を十分に取り、情報を詰め込みすぎない
- 1スライド1メッセージを原則とする

## カラーパレット
- メインカラー：#1E3A5F（ネイビー）
- アクセントカラー：#2E86AB（ブルー）
- テキスト：#1A1A1A（ほぼ黒）
- 背景：#FFFFFF（白）または #F7F9FC（薄いグレー）

## フォント
- 見出し：太字・大きめ（相当フォント: 28〜36pt）
- 本文：読みやすいサイズ（14〜18pt）
- 強調：アクセントカラーを使用

## レイアウトルール
- タイトルスライドには会社名・日付・担当者名を含める
- 各スライドに見出しを必ず入れる
- データや数字はグラフ・図表で視覚化する
- 箇条書きは1スライドにつき最大5項目まで

## アイコン・画像
- シンプルなフラットアイコンを使用
- 装飾的な画像は最小限に留める

## スライド構成テンプレート
1. 表紙（タイトル・会社名・日付）
2. 目次 / アジェンダ
3. 課題・背景
4. 提案内容
5. 期待される効果・メリット
6. スケジュール・ロードマップ
7. お見積もり / 投資対効果
8. まとめ・ネクストステップ

## 作業指示
- 上記スタイルガイドに従ってスライドを作成すること
- アップロードされたファイルの内容を忠実にスライドに反映すること
- 日本語で作成すること
